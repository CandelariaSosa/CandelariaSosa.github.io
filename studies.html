¿Y la experiencia?

Aunque aún no cuento con experiencia laboral formal como programadora o analista, mi formación en matemática me ha dado una base sólida para desarrollar el pensamiento lógico y la capacidad de resolver problemas, habilidades fundamentales en el mundo del desarrollo de software.
Además, mi trayectoria como docente me ha permitido trabajar en equipo con otros profesionales, gestionar grupos diversos de estudiantes y comunicarme de manera clara y efectiva, lo cual considero un valor agregado en cualquier entorno de trabajo.
Este sitio web fue creado desde cero como parte de mi proceso de aprendizaje. Fue desarrollado utilizando HTML, CSS, Bootstrap, y es completamente responsive, es decir, se adapta a distintos dispositivos. También trabajé con Git y GitHub para el control de versiones y la gestión del código.
